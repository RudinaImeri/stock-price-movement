# Stock Price Movement Prediction (ML + Streamlit)

An interactive **machine learning dashboard** that predicts stock price movements (**Buy / Hold / Sell**) using historical market data fetched from **Yahoo Finance (yfinance)**.

Built with:
- Machine Learning (scikit‑learn)
- Streamlit Dashboard
- Live‑like market data via yfinance

---

## Features

- **Data Overview**
  - Fetch stock market data from Yahoo Finance
  - Preview cleaned & processed data

- **Model Training**
  - Train ML model on selected stock
  - Automatic feature encoding
  - Accuracy & F1‑score evaluation

- **Feature Importance**
  - Visualize most important features used by the model

- **Predictions Dashboard**
  - Choose stocks (Apple, Tesla, Nvidia, etc.)
  - Buy / Hold / Sell predictions
  - Confidence scores
  - Price, change, volume
  - Styled like a real trading platform

---

## Tech Stack

- Python 3.10+
- Streamlit
- Pandas / NumPy
- Scikit‑learn
- yfinance
